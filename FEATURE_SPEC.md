# New Data Structure Features Documentation

Date: 2026-04-05 11:21:23 UTC

## Introduction
This document outlines the new data structure features that will be implemented in the `vaultfs-core` repository. The features will focus on various fundamental concepts of data structures, including Searching, Sorting, Linked List, ArrayList, Graphs, Hash Tables, Stacks, Strings, Queues, Trees, and Binary Trees.

## Features

### 1. Searching Algorithms
- **Linear Search**: A straightforward searching technique that checks each element in a list until the target element is found.
- **Binary Search**: An efficient algorithm that divides the sorted array in half, eliminating half of the remaining elements with each iteration.

### 2. Sorting Algorithms
- **Bubble Sort**: A simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.
- **Quick Sort**: An efficient sorting algorithm that uses divide-and-conquer principles to sort elements.

### 3. Linked List Implementation
- **Singly Linked List**: A data structure that consists of nodes where each node contains a data field and a link to the next node.
- **Doubly Linked List**: Similar to the singly linked list but with links to both the previous and next nodes.

### 4. ArrayList
- A resizable array implementation that allows for dynamic array size and provides methods for adding, removing, and accessing elements.

### 5. Graph Structures
- **Graph Representation**: Implementation of graphs using adjacency lists and matrices. Methods for adding/removing vertices and edges.
- **Graph Algorithms**: Depth First Search (DFS) and Breadth First Search (BFS) for graph traversal.

### 6. Hash Tables
- A data structure that implements an associative array abstract data type, a structure that can map keys to values.
- Implementation of hash functions to ensure key-value uniqueness.

### 7. Stacks
- A collection of elements that follows the Last In First Out (LIFO) principle. Implementation includes push, pop, and peek operations.

### 8. Strings
- Implementation of common string operations, including concatenation, substring search, and string manipulation functionalities.

### 9. Queues
- A collection that follows the First In First Out (FIFO) principle. Implementation includes enqueue, dequeue, and front operations.

### 10. Tree Structures
- **Binary Tree**: A tree data structure in which each node has at most two children, referred to as the left child and the right child.
- **Binary Search Tree**: A special type of binary tree where the left child's key is less than the parent's key, and the right child's key is greater.

## Conclusion
This outline serves as a foundational framework for the design and implementation of the new data structures into the `vaultfs-core` repository.
